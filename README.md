> # numarket

Flutter app for Nubank test

> ## Usage
#### Installation
```bash
$ flutter pub get
```
#### Run tests
```bash
$ flutter test
```
#### Run app in device
```bash
$ flutter run
```
### Development
#### Build stores
```bash
$ flutter packages pub run build_runner watch --delete-conflicting-outputs
```
